Системные требования

Windows 10 версии 17134 и выше. Наличие установленного приложения App Installer для установки не из Windows Store.

Основные возможности

Доступ к обширному каталогу релизов с возможность фильтрации и сортировки по различным критериям. Встроенный онлайн видеоплеер для просмотра контента с возможностями перемотки, переходов а также проецирования видеоконтента на устройства поддерживающие эту технологию (телевизоры, проекторы, телеприставки и т.п.). Просмотр комментариев к релизам. Поддержка различных устройств от планшетов до Xbox One S/X с различными устройствами ввода.

Windows Store

Установка через Windows Store

Чтобы установить приложение через Windows Store Вам достаточно будет перейти на страницу приложения и нажать на кнопку Получить после чего нажать на кнопку Установить. Обновления будут автоматически применяться согласно настройкам системы

Web-инсталлятор

Установка через Web-инсталлятор

Вы можете установить приложение с помощью web-инсталлятора. Это специальная программа которая выполнит все необходимые шаги для установки за Вас. Сама программа будет запущена во время инсталляции, но может быть запущена повторно для обновления приложения. Иконка на рабочем будет называться Anilibria Installer. Если Вы уже поставили приложение то обновить его можно и по этой ссылке.
