Клиент сайта AniLibria.TV для Windows 10
Перейти на сайт AniLibria.TV
Системные требования
Windows 10 версии 17134 и выше. Наличие установленного приложения App Installer для установки не из Windows Store.
Основные возможности
Доступ к обширному каталогу релизов с возможность фильтрации и сортировки по различным критериям. Встроенный онлайн видеоплеер для просмотра контента с возможностями перемотки, переходов а также проецирования видеоконтента на устройства поддерживающие эту технологию (телевизоры, проекторы, телеприставки и т.п.). Просмотр комментариев к релизам. Поддержка различных устройств от планшетов до Xbox One S/X с различными устройствами ввода.
Варианты установки
Следует также знать что все предложенные варианты взаимоисключают друг друга т.е. Вы не можете поставить их одновременно!
  • Установка через Windows Store
  • Установка с помощью web-инсталлятора (приложение будет установлено не через Windows Store, требуется наличие App Installer)
  • Установка с помощью offline-инсталлятора (приложение будет установлено не через Windows Store)
  • Установка тестовых версий (для продвинутых пользователей)
Далее будет описан каждый из этих вариантов в подробностях.
Установка через Windows Store
 Чтобы установить приложение через Windows Store Вам достаточно будет перейти на страницу приложения и нажать на кнопку Получить. Обновления будут автоматически применяться согласно настройкам системы, обычно автообновление включено по умолчанию. Для установки достаточно нажать на бейдж ниже на устройстве с Windows 10:

Установка с помощью web-инсталлятора (приложение будет установлено не через Windows Store)
 Вы можете установить приложение с помощью web-инсталлятора. Это специальная программа которая выполнит все необходимые шаги для установки за Вас. Сама программа будет запущена во время инсталляции, но может быть запущена повторно для обновления приложения. Иконка на рабочем будет называться Anilibria Installer. Скачать инсталлятор можно по этой ссылке. Если Вы уже поставили приложение то обновить его можно и по этой ссылке.
Установка с помощью offline-инсталлятора (приложение будет установлено не через Windows Store)
 Этот вид установки специально сделан для версиий Windows в которых нет Windows Store и нет App Installer. Инсталляторы будут доступны для каждой стабильной версии и будет распространяться на этой странице. Единственный недостаток этого вида установки связан с тем что автообновление приложения здесь не предусмотрено. Вам прийдется самостоятельно отслеживать новые версии и ставить их с помощью инсталлятора для новых версий.
Установка тестовых версий (для продвинутых пользователей)

Если у Вас есть лишнее время, желание и пара девайсов или виртуальных машин с windows 10 Вы можете помочь с тестированием новых версий приложения до их выпуска.
Есть подробная инструкция для установки тестовых версий по этой ссылке.
